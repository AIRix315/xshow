# XShow

节点式可视化 AI 工作流编辑器 — Chrome 扩展

通过拖拽节点、连接数据流，快速搭建多步骤 AI 创作流程。

## 功能特性

- **节点编辑器** — 30+ 内置节点类型，拖拽连线构建工作流
- **多协议支持** — OpenAI / Gemini / Anthropic / ComfyUI / RunningHub
- **生成节点** — 图片、视频、文本（LLM）、音频（TTS）、3D 模型
- **处理节点** — 裁剪、标注、帧提取、视频裁剪/拼接、宫格拆分/合并
- **万能节点（OmniNode）** — 自定义 HTTP API，支持同步/异步轮询
- **RunningHub 集成** — APP 快捷创作、工作流执行、ZIP 解压
- **项目管理** — 多项目隔离、.xshow 导入/导出、增量保存
- **资源中转** — 跨页面资源收集与复用

## 安装

1. 从 [Releases](https://github.com/AIRix315/XShow/releases) 下载最新 `xshow-v*.zip`
2. 解压到本地目录
3. Chrome → `chrome://extensions/` → 开启「开发者模式」
4. 点击「加载已解压的扩展程序」→ 选择解压目录
